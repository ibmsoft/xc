印度尼西亚人口统计--echarts3.0 图表。
数据来源 https://www.wikiwand.com/zh/印度尼西亚行政区划。
echarts3.0自定义地图构造 http://geojson.io 。
-------------------------修改显示图表说明------------------
双击indonesia.html即可看到图表效果，若chrome浏览器下不能显示图表，
  请设置浏览器允许跨域浏览，操作方式： 在chrome浏览器的属性窗口中，在【快捷方式】页签的【目标】栏后加上 --allow-file-access-from-files
 
 请将数据文件放在与当前页面平行的data文件夹中，并且命名为indonesia.josn。
 默认样式为 vintage，可选的样式为 dark、vintage、macarons和shine，
 若要设置样式，请修改indonesia.json中的thema属性的值，
 若不需要设置背景样式，请将indonesia.json中的thema属性值设成"-1"。
 请手动设置indonesia.json中的所包含数据的最小值 minValue 和 最大值 maxValue
